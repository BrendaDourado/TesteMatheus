Mesclando códigos (merge)

O merge é uma função que utilizamos para adicionar todas as funcionalidades de uma branch em outra. O comando é esse: git merge nome_da_branch. É comum ocorrer erros na tentativa de realizar o merge ou o pull e cabe a você, dev, resolver.
