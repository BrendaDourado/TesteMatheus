Existem algumas formas de iniciar um projeto utilizando o Git. Uma delas é você mesmo criar um repositório no GitHub para que seus colegas utilizem dele para adicionar atualizações ao projeto.

Para isso você deve possuir uma conta no GitHub, criar um repositório na configuração público. Após criar insira um arquivo index com algum texto dentro e após isso compartilhe o link dele para que seus companheiros possam clonar sua pasta e essa é a segunda forma de iniciar o projeto.

Se não foi você quem criou o repositório você deve realizar os seguintes passos:

  1. Criar uma pasta em sua máquina;
  2. Clicar sobre ela com o botão direito;
  3. Clicar em Git Bash Here;
  4. Dentro do terminal do git você ira digitar o seguinte código:
    git clone link_do_repositório
