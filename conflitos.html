Lidando com conflitos

Conflitos são os famosos problemas que irão acontcer durante as nossas tentativas de pul, push, merge ou basicamente qualquer comando que tentarmos por no git.

Cada conflito tem um jeito de resolver, algumas das soluções possíveis são:

  • Resolver o problema de uma página manualmente, alterando o seu conteúdo e após finalizar use o seguinte: git add OU git add nome_do_arquivo
  • Você pode remover o arquivo que está dando problema com o comando: git rm nome_do_arquivo
  • As vezes é necessário aceitar atualizações vindas de agentes externos. Para habilitar essa opção use o comando: git checkout --theirs nome_do_arquivo
  • Também é possível alterar a configuração para aceitar alteração local com o comando: git checkout --ours nome_do_arquivo
  • Caso tudo dê errado você pode usar o git merge --abort ele irá cancelar a mesclagem. Depois disso tente do zero.
